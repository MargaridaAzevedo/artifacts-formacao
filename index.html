🔍 Conceito
O que é um Artifact?
A diferença entre texto e magia — com o mesmo prompt
📝

Sem Artifacts

O Claude responde em texto. Muito bom texto — planificações, fichas, e-mails. Útil, mas estático. Não é interativo, não é projetável, não cativa os alunos.

✨

Com Artifacts ativado

O mesmo prompt gera um painel interativo à direita — quiz com feedback imediato, simulação animada, dashboard clicável. Renderiza HTML, JavaScript, React e SVG.

⚙️ Para ativar: claude.ai → ícone ✦ (canto inferior direito) → "Use Artifacts". Disponível na conta Pro. Feito isto, qualquer prompt que justifique conteúdo interativo gera automaticamente um Artifact.

Ato 3 — O Professor Invisível

Prancha CAA com Síntese de Voz

Sentimentos, Necessidades, Sala de Aula e Refeições. Clica e o browser fala em português europeu.
